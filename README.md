# EMU8086 Assembly Oyun Projesi

Bu proje, 8086 Assembly dili ile gelistirilmis metin tabanli bir kacis oyunudur. Temel amac, oyuncu karakterini (A) kontrol ederek surekli takipte olan dusman karakterinden (X) kacmaktir. 

## Gereksinimler

Programin calisabilmesi icin bilgisayarinizda EMU8086 emulatorunun kurulu olmasi gerekmektedir.

## Kontroller

- Hareket Etme: W, A, S, D tuslari veya Yon Tuslari
- Can kaybindan sonra devam etme: Bosluk (Space) tusu

## Oynanis Detaylari

- Oyuna 3 can ile baslarsiniz.
- Dusmanin (X) hareket hizi zaman gectikce otomatik olarak artar.
- Dusmana temas ettiginizde 1 can kaybedersiniz, oyun duraklar ve pozisyonlar sifirlanir.
- Oyun zorlugunu dengelemek amaciyla, caniniz azaldikca sizin de hareket hiziniz artar.
- Ekranin ust kisminda caninizi ve hiz seviyelerini anlik olarak gorebilirsiniz.

## Nasil Calistirilir?

1. Projedeki Assembly kodunu (.asm veya .txt icerisindeki) kopyalayin.
2. EMU8086 programini calistirin.
3. Yeni bir belge acarak (COM template) kodu icerisine yapistirin.
4. Ust menuden "Emulate" butonuna tiklayin.
5. Acilan pencerede "Run" butonuna basarak oyunu oynayabilirsiniz.
